# dog-age-analysis
Small Python project I built on my dog's birthday.

## Objective
Compare different methods to estimate a dog's age in human years and visualize results.

## Methods
- Simple rule: 1 dog year = 7 human years.
- More realistic model:
- Year 1 --> 15 years
- Year 2 --> 24 years
- Them +5 years per additional year

## Tech stack
- Python
- pandas
- matplotlib

## Output
The plot shows how the two methods diverge over time.
